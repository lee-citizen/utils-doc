## 启动前的依赖

### 安装node

> docsify需要node环境，node自带npm


### 安装docsify

```bash
npm i docsify-cli -g
```


## 启动模板

在`docsify`目录下启动模板：

```bash
# 打开下载的地址

docsify serve docs
```
访问地址:`http://localhost:3000`
